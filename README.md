# MoneyMitra 360 – Agentic AI Banking Companion

A high-fidelity, clickable React + Tailwind frontend prototype for a National Level Banking AI Hackathon.

## What this prototype includes

- Landing page with fintech hero section
- AI chat screen with recommendation cards
- Customer profile screen
- Product recommendation screen
- Digital adoption dashboard
- Onboarding journey tracker
- Banker analytics dashboard
- Customer detail screen
- AI explainability popup
- Success screen
- Bottom navigation and floating AI assistant
- Dummy JSON data only
- No backend, no APIs, no authentication

## Tech stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React icons

## Run locally

```bash
npm install
npm run dev
```

Then open the local Vite URL in your browser.

## Main prototype flow

Landing Page → AI Chat → Customer Profile → Recommendation → Journey Tracker → Banker Dashboard → Customer Details → Success Screen

## Notes

This is intentionally frontend-only and uses static dummy data from `src/data/mockData.js`.
